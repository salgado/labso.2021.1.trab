Instruções para a prova
--------------------------

1 - Faça um login no seu github.com
2 - Faça um FORK do repositório https://github.com/salgado/labso.2019.1.av1.git 
3 - Clonar o novo repósitório criado na sua máquina local
4 - Entrar na pasta labso.2019.1.av1
5 - Abrir o arquivo prova1-2019.txt e iniciar a prova (sugestão: deixe esse arquivo aberto em outra janela)


Instruções básicas do github
----------------------------
- clonar repositório
git clone <url-do-seu-repositorio>

- submeter arquivo da sua pasta
git add <arquivo>
git commit -m "minha mensagem"
git push

- (Somente se preciso for) sincronizar sua pasta com github
git pull
